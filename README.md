#如何贡献

点击右上角的 "Fork" 按钮，然后，

`
#yourName 为您的github昵称
git clone https://github.com/yourName/apicloud_docs.git
`

修改完毕后，提交到您自己的仓库，然后发起 Pull Requests，我们定期会审核一下，没问题的修改，我们会 merge 进来。
